Mall Ticketing System

Overview:

This project is a database-driven ticketing system designed to manage event bookings within a mall environment.

Problem:

Manual ticketing processes can lead to inefficiencies, long queues, and poor record management.

Solution:

This system automates ticket booking, customer management, and payment tracking using a structured database.

System Features:

* Customer registration
* Ticket booking
* Payment tracking
* Event management

Database Design:

The system uses relational tables such as:

* Customers
* Tickets
* Events
* Payments

Process Flow:

1. Customer selects event
2. Booking is created
3. Payment is processed
4. Ticket is confirmed

Technologies Used:

* SQL Database
* Visual Studio

Key Learnings:

* Database design and relationships
* Business process automation
* System thinking and structure
